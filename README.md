

This project was not just another assignment for me — it was a journey full of learning, frustration, debugging, failures, and growth.

I fine-tuned the Qwen Qwen2-VL-2B-Instruct model for **Document Image → Markdown Generation** and built the complete multimodal pipeline from scratch.

The project included:

• Dataset preprocessing
• LoRA fine-tuning
• 4-bit quantization
• Custom Dataset & DataLoader
• Prompt engineering
• Zero-shot vs fine-tuned evaluation
• Gradio demo interface

There were many moments where things simply refused to work.

GPU memory crashes, preprocessing errors, broken image samples, training failures, dependency issues, and endless debugging sessions made this project genuinely challenging.

Sometimes a single error took hours to solve.

But step by step, problem by problem, things finally started working — and seeing the model successfully generate structured Markdown from document images made every struggle worth it.

More than the final output, this project taught me patience, consistency, and how real AI development actually feels behind the scenes.

Through this journey, I gained hands-on experience with:

• Vision-Language Models (VLMs)
• Multimodal AI systems
• LoRA & PEFT fine-tuning
• Quantization techniques
• Prompt engineering
• Memory-efficient training pipelines

This assignment gave me much more than technical knowledge — it gave me confidence that I can build and understand advanced AI systems with persistence and continuous learning.

Grateful for the experience and excited to keep learning 🚀
